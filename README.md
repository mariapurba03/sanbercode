# Automation Testing for Orange HRM Demo Website Using Cypress

## Project Overview
This project is an automation testing implementation for the OrangeHRM demo website using Cypress.
The purpose of this project is to validate core functionalities and ensure application reliability
through automated end-to-end testing.

## Tools & Technologies
- Cypress
- JavaScript
- Page Object Model (POM)
- OrangeHRM Demo Website

## Test Scope
- Login functionality
- Forgot Password functionality
- Directory feature

## Project Structure
- cypress/e2e : Test cases
- cypress/fixtures : Test data
- cypress/support/pom : Page Object files
- cypress.config.js : Cypress configuration
